Cara Penggunaan

Sebelum menginstall module2 yang dibutuhkan anda harus memakai NodeJs v20.11.0



Disini saya menggunakan Nuxt v3 sebagai frontend dan Express js sebagai middleware api dari https://www.mmobomb.com/api1/games?platform=pc dan https://www.mmobomb.com/api1/game?id=[game_id]

Sehingga setiap Rest API yang akan di gunakan frontend dari pihak ke tiga akan di ambil melalui express js sehingga nanti di frontend akan menggunakan ip localhost untuk pemanggilan rest api nya.


1. Install Module & Jalankan Server

## pindah ke folder backend-api-express
cd /backend-api-express

## install module express js
npm install

## jalankan service dari express js
node index.js


## pindah ke folder frontend
cd /frontend

## install nuxt js
npm install

## jalankan server frontend
npm run dev

2. Url Aplikasi

  1. Backend Url

    http://127.0.0.1:3001/api/data
    digunakan untuk mengambil data dari https://www.mmobomb.com/api1/games?platform=pc



    http://127.0.0.1:3001/api/data/:id
    digunakan untuk mengambil data detail dari game https://www.mmobomb.com/api1/game?id=[game_id]



  2. Frontend Url

    http://localhost:3000/
    digunakan sebagai halaman utama landing page dari card game yang ada di Rest Api



    http:/localhost:3000/detail/:id
    digunakan untuk menampilkan detail dari game yang ada di aplikasi
